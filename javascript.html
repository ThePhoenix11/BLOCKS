  • It provides the features to work on both frontend and backend operations.

GETTING STARTED
Learning JS can be summarized in 3 paths :
  1. Learning the basics of JS - syntax, varibles and data types.
  2. Understanding the basic frameworks and building the projects.
  3. Learning the advanced concepts, packages and deploying the websites.
Tips & Tricks :
For basic JS, work on projects like rock,paper & scissors game.
